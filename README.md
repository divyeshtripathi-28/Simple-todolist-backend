# To Do List (Backend)

This project is a To-Do List web application that uses an SQL database to store and manage tasks. The application allows users to add, view, update, and delete tasks, demonstrating the integration of a database with a web application.

The project focuses on implementing CRUD operations (Create, Read, Update, Delete) using an SQL database while providing a simple and intuitive user interface.

# Features
	•	Add new tasks
	•	View all tasks
	•	Mark tasks as completed
	•	Edit existing tasks
	•	Delete tasks
	•	Persistent data storage using SQL database
